Settings

Cannot change while a game is running.

- iPhone uses pre-recorded English letter sounds (works in home-screen app).
- Press only when the current stimulus matches exactly N trials ago.

How to play

Dual N-Back: remember position (visual) and sound (auditory) at the same time.

  • POSITION: tap when the current position matches N trials ago
  • SOUND: tap when the current sound matches N trials ago
  • START / STOP: center button toggles start, pause, and resume
  • INFO: show or hide round, N, trial, and score
  • The first N trials have no matches.
Scoring
Correct +2, wrong −1, miss −1
