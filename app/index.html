How to play

Dual N-Back: remember position (visual) and sound (auditory) at the same time.

  • Position: press L when the current cell matches N trials ago
  • Sound: press A when the current letter matches N trials ago
  • Space: start or stop the round (same as the START button)
  • INFO: show or hide round, N, trial, and score
  • Visual and audio matches are independent each trial.
Scoring
Correct hit +2, false alarm −1, miss −1.
